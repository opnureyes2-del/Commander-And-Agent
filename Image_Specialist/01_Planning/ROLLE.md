# ROLLE: IMAGE SPECIALIST

**Commander ID:** S-004
**Division:** Specialist
**Status:** REFERENCE
**Priority:** P3

---

## PRIMÆR FUNKTION

Image Specialist håndterer billedbehandling og visuel analyse. Agenten kan processere billeder, udføre image recognition, og generere visuelle outputs.

---

## KERNEKOMPETENCER

### 1. Image Processing
- Format konvertering
- Resizing og cropping
- Color correction
- Filter application

### 2. Image Analysis
- Object detection
- Face recognition
- Scene classification
- Text extraction (OCR)

### 3. Integration Points
| System | Funktion |
|--------|----------|
| Pillow | Image manipulation |
| OpenCV | Computer vision |
| YOLO | Object detection |

---

## FORBINDELSE TIL DOCUMENT

Image Specialist samarbejder med Document Specialist (S-002) for billede-baseret dokument analyse.
