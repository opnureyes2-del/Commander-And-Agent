# ROLLE: DATA COMMANDER

**Commander ID:** M-004
**Division:** Mobile
**Status:** PLACEHOLDER
**Priority:** P3

---

## PRIMÆR FUNKTION

Data Commander er den databehandlende agent der håndterer dataprocessering, analyse og transformation. Agenten fungerer som bindeled mellem rå data og actionable insights.

---

## KERNEKOMPETENCER

### 1. Data Processing
- Parse og transform forskellige dataformater
- CSV, JSON, XML håndtering
- Data validation og sanitering
- Batch processing

### 2. Data Analysis
- Statistical analysis
- Pattern recognition
- Trend identification
- Anomaly detection

### 3. Data Transformation
- Format conversion
- Data aggregation
- Filtering og sorting
- Join og merge operations
