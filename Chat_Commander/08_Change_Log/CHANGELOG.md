## CHANGE REQUEST PROCESS

### How to Request Changes

1. Document the change in TODO.md
2. Assign task ID following format: TASK [Phase].[Order].[Task#]
3. Update STATUS.md when work begins
4. Log progress in DEV_LOG.md
5. Record change in CHANGELOG.md upon completion
6. Update TEST_RESULTS.md with test coverage

### Change Categories

| Category | Examples | Approval Required |
|----------|----------|-------------------|
| Documentation | README updates, comments | No |
| Minor Feature | New helper function | No |
| Major Feature | New capability | Yes |
| Breaking Change | API modification | Yes |
| Security | Any security change | Yes |
